## 2. 步进电机 & 5. 电机

本节介绍智能物流小车的底盘电机控制，包括：
- 直流减速电机（4轮麦克纳姆轮）
- 编码器测速
- PID 速度闭环控制
- 位置控制（脉冲计数）

### 2.1 硬件资源分配

| 电机 | PWM 输出 | 编码器接口 | 方向控制 |
|------|----------|------------|----------|
| 电机A (右前) | TIM10 CH1, TIM11 CH1 | TIM2 | GPIO |
| 电机B (左前) | TIM9 CH1, TIM9 CH2 | TIM3 | GPIO |
| 电机C (左后) | TIM1 CH1, TIM1 CH2 | TIM4 | GPIO |
| 电机D (右后) | TIM1 CH3, TIM1 CH4 | TIM5 | GPIO |

> 💡 **设计思路**：使用双通道 PWM（H桥）控制电机正反转，编码器使用正交解码模式。

### 2.2 核心代码结构

#### 2.2.1 电机 PWM 初始化 (Motor.c)

```c
void Motor_Init(void)
{
    // 启动电机A (右前) 的PWM通道
    HAL_TIM_PWM_Start(&htim10, TIM_CHANNEL_1);
    HAL_TIM_PWM_Start(&htim11, TIM_CHANNEL_1);
    // ... 其他电机类似
}
```

**精华解读**：
- 使用两个 TIM 通道产生互补 PWM，分别控制 H 桥的左右两侧
- 正转时：A通道输出 PWM，B通道输出 0
- 反转时：A通道输出 0，B通道输出 PWM

#### 2.2.2 编码器速度计算 (encoder.c)

```c
// 速度转换系数
static const float VELOCITY_COEFFICIENT = (60.0f * CONTROL_FREQUENCY) / ENCODER_RESOLUTION;

void Encoder_Update(Encoder_t *encoder)
{
    // 1. 读取当前硬件计数器值
    int32_t current_counter = __HAL_TIM_GET_COUNTER(encoder->htim);
    
    // 2. 计算脉冲增量，处理溢出
    int32_t delta = current_counter - encoder->Last_counter;
    
    // 3. 处理 16 位定时器溢出
    if (encoder->htim->Instance == TIM3 || encoder->htim->Instance == TIM4) {
        if (delta > 32767) delta -= 65536;
        else if (delta < -32768) delta += 65536;
        delta = -delta; // 左轮符号翻转
    }
    
    // 4. 更新累计脉冲和速度
    encoder->Total_counter += delta;
    encoder->Velocity = (float)delta * VELOCITY_COEFFICIENT;
}
```

**精华解读**：
- 编码器脉冲 → RPM 转换公式：`(增量脉冲 / 分辨率) * (采样频率 * 60)`
- 处理计数器溢出是测速的关键！

#### 2.2.3 PID 控制核心 (Motor_PID.c)

```c
// PID 参数结构体
typedef struct {
    float Kp, Ki, Kd;
    float Target;      // 目标值
    float Measured;    // 实际值
    float Error;       // 偏差
    float PrevError;   // 上次偏差
    float Output;      // PID 输出
    float Integral;    // 积分累计
} PID_Controller_t;

// 增量式 PID 计算
float PID_Calculate(PID_Controller_t *pid, float target, float measured)
{
    pid->Target = target;
    pid->Measured = measured;
    pid->Error = target - measured;
    
    // 增量计算
    float p_term = pid->Kp * (pid->Error - pid->PrevError);
    float i_term = pid->Ki * pid->Error;
    float d_term = pid->Kd * (pid->Error - 2*pid->PrevError + pid->PrevPrevError);
    
    pid->Output += p_term + i_term + d_term;
    
    // 更新历史值
    pid->PrevPrevError = pid->PrevError;
    pid->PrevError = pid->Error;
    
    return pid->Output;
}
```

**精华解读**：
- 使用**增量式 PID**，输出是 PWM 变化量，稳定性更好
- 位置模式：通过累计脉冲数控制行驶距离
- 速度模式：实时速度闭环

#### 2.2.4 10ms 定时器中断 (main.c)

```c
// TIM6 配置为 10ms 中断 (100Hz)
void HAL_TIM_PeriodElapsedCallback(TIM_HandleTypeDef *htim)
{
    if (htim->Instance == TIM6) // 10ms 中断
    {
        // 1. 更新所有编码器数据
        Encoders_All_Update();
        
        // 2. PID 计算
        Motor_PID_Update_All();
        Yaw_PID_Update(); // 转向环
    }
}
```

**精华解读**：
- PID 控制频率固定为 100Hz，避免控制振荡
- 中断中只做计算，不做阻塞操作

### 2.3 常见问题与改进建议

| 问题 | 原因 | 改进建议 |
|------|------|----------|
| 电机抖动 | PID 参数未调好 | 使用 Z-N 整定法或试凑法 |
| 停车不平滑 | 缺少平滑启动/刹车 | 添加斜坡函数 (Ramp/Slew Rate) |
| 编码器计数不准 | 计数器溢出处理 bug | 使用 32 位定时器或软件补偿 |
| 转向不稳定 | 差速计算有问题 | 检查左右轮速度符号 |
