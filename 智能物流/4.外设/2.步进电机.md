### 2.1 电机驱动原理

#### 2.1.1 H 桥驱动电路

直流电机转向控制采用 H 桥驱动拓扑，其电路结构如下：

```
         +Vcc
           |
      +----+----+
      |         |
    +-+-       +-+-
    |A|       |B|
    +-+-       +-+-
      |         |
      +----+----+
           |
         Motor
           |
      +----+----+
      |         |
    +-+-       +-+-
    |C|       |D|
    +-+-       +-+-
      |         |
      +----+----+
           |
          GND
```

**控制逻辑表**：

| A | B | C | D | 电机状态 |
|---|---|---|---|----------|
| 1 | 0 | 0 | 1 | 正转 |
| 0 | 1 | 1 | 0 | 反转 |
| 1 | 1 | 0 | 0 | 制动 |
| 0 | 0 | 0 | 0 | 悬空 |

**物理意义**：通过控制四个 MOSFET 的导通组合，改变电机两端电压极性，从而改变转动方向。

#### 2.1.2 PWM 调速原理

STM32 通过 TIM 输出 PWM（脉宽调制）信号控制 H 桥：

```c
// PWM 周期计算公式：
// Period = (PSC+1) × (ARR+1) / 时钟频率
// 例：PSC=83, ARR=999, 时钟=84MHz
// 周期 = 84 × 1000 / 84MHz = 1ms → 1kHz

/**
 * @brief 设置电机A转速
 * @param speed: 正值=正转，负值=反转，绝对值=速度大小
 */
void MotorA_SetSpeed(int speed)
{
    if (speed >= 0)  // 正转
    {
        // TIM10 输出 PWM 信号到 H 桥 A 侧
        __HAL_TIM_SET_COMPARE(&htim10, TIM_CHANNEL_1, speed);
        // TIM11 输出 0，H 桥 B 侧关闭
        __HAL_TIM_SET_COMPARE(&htim11, TIM_CHANNEL_1, 0);
    }
    else  // 反转
    {
        __HAL_TIM_SET_COMPARE(&htim10, TIM_CHANNEL_1, 0);
        __HAL_TIM_SET_COMPARE(&htim11, TIM_CHANNEL_1, -speed);
    }
}
```

**PWM 占空比公式**：
```
占空比 = CCR / ARR × 100%
```

占空比与电机转速近似呈线性关系，占空比越大，电机两端平均电压越高，转速越快。

### 2.2 编码器测速

#### 2.2.1 正交编码器原理

增量式编码器输出两相正交脉冲信号 A 和 B，相位差 90°：

- **A 相超前 B 相 90°** → 正转
- **B 相超前 A 相 90°** → 反转

STM32 TIM 编码器模式自动识别方向并计数，无需软件判断方向。

```c
/**
 * @brief 编码器初始化
 * @param encoder: 编码器结构体指针
 * @param htim: 对应定时器句柄
 */
void Encoder_Init(Encoder_t *encoder, TIM_HandleTypeDef *htim)
{
    encoder->htim = htim;
    encoder->Last_counter = 0;
    encoder->Total_counter = 0;
    encoder->Velocity = 0.0f;

    // 启动编码器模式
    HAL_TIM_Encoder_Start(htim, TIM_CHANNEL_ALL);
    // 计数器清零
    __HAL_TIM_SET_COUNTER(htim, 0);
}
```

#### 2.2.2 速度计算

```c
// 速度转换系数推导：
// RPM = (脉冲增量 / 分辨率) × (每秒采样次数 × 60秒)
// 分辨率 = 编码器线数 × 4 (四倍频)
// 采样频率 = CONTROL_FREQUENCY (100Hz)
static const float VELOCITY_COEFFICIENT = (60.0f * CONTROL_FREQUENCY) / ENCODER_RESOLUTION;

/**
 * @brief 更新编码器数据
 * @param encoder: 编码器结构体指针
 * @note 在 TIM6 中断中每 10ms 调用一次
 */
void Encoder_Update(Encoder_t *encoder)
{
    // 1. 读取当前计数值（TIM 计数器自动累加）
    int32_t current = __HAL_TIM_GET_COUNTER(encoder->htim);

    // 2. 计算与上次的中脉冲增量（处理回绕）
    int32_t delta = current - encoder->Last_counter;

    // 3. 16位计数器溢出处理（回绕检测）
    // 计数器从 65535 回绕到 0 为上溢出
    // 计数器从 0 回绕到 65535 为下溢出
    if (encoder->htim->Instance == TIM3 || encoder->htim->Instance == TIM4)
    {
        if (delta > 32767)       // 正向溢出：10 → 65530
            delta -= 65536;
        else if (delta < -32768) // 负向溢出：65530 → 10
            delta += 65536;
        
        // 4. 左轮方向反转（机械安装导致）
        delta = -delta;
    }

    // 5. 更新累计脉冲数（用于位置控制）
    encoder->Total_counter += delta;

    // 6. 计算瞬时速度 RPM
    encoder->Velocity = delta * VELOCITY_COEFFICIENT;
}
```

### 2.3 PID 控制算法

#### 2.3.1 增量式 PID

位置式 PID 输出为绝对值，增量式 PID 输出为控制量的变化量：

```c
/**
 * PID 控制结构体
 */
typedef struct {
    float Kp;                    // 比例系数
    float Ki;                    // 积分系数
    float Kd;                    // 微分系数
    float Target;                // 目标值
    float Measured;              // 测量值
    float Error;                 // 当前误差
    float PrevError;            // 上一次误差
    float PrevPrevError;         // 上上次误差
    float Output;                // PID 输出
} PID_Controller_t;

/**
 * @brief 增量式 PID 计算
 * @param pid: PID 结构体指针
 * @param target: 目标值
 * @param measured: 测量值（实际值）
 * @return PID 输出值
 * 
 * 增量式 PID 公式：
 * Δu = Kp×(e[k]-e[k-1]) + Ki×e[k] + Kd×(e[k]-2e[k-1]+e[k-2])
 * u[k] = u[k-1] + Δu
 */
float PID_Calculate(PID_Controller_t *pid, float target, float measured)
{
    // 更新目标值和测量值
    pid->Target = target;
    pid->Measured = measured;
    
    // 计算当前误差
    pid->Error = target - measured;

    // P 项：比例环节，反映当前误差
    float p_term = pid->Kp * (pid->Error - pid->PrevError);

    // I 项：积分环节，消除稳态误差
    float i_term = pid->Ki * pid->Error;

    // D 项：微分环节，预测误差趋势
    float d_term = pid->Kd * (pid->Error - 2*pid->PrevError + pid->PrevPrevError);

    // 累加增量得到输出
    pid->Output += p_term + i_term + d_term;

    // 更新误差历史
    pid->PrevPrevError = pid->PrevError;
    pid->PrevError = pid->Error;

    return pid->Output;
}
```

#### 2.3.2 PID 参数调节

| 参数 | 作用 | 调节方法 |
|------|------|----------|
| Kp | 快速响应误差 | 增大 → 响应快、超调大；减小 → 稳、慢 |
| Ki | 消除稳态误差 | 增大 → 精度高、可能振荡；减小 → 稳态精度低 |
| Kd | 阻尼预测 | 增大 → 抑制振荡；减小 → 可能振荡 |

**Ziegler-Nichols 整定法**：
1. 置 Ki=Kd=0
2. 逐渐增大 Kp 至系统振荡
3. 记录临界振荡 Kc 和周期 Pc
4. 按公式计算：Kp=0.6Kc, Ki=1.2Kc/Pc, Kd=0.075Kc×Pc

### 2.4 定时器中断

#### 2.4.1 TIM6 中断机制

PID 控制需要固定采样周期，本系统使用 TIM6 产生 10ms 定时中断（100Hz）：

```c
// TIM6 配置参数：
// PSC = 8399, ARR = 99
// 中断周期 = (8399+1) × (99+1) / 84MHz = 10ms
// 中断频率 = 1 / 10ms = 100Hz

/**
 * @brief 定时器周期中断回调
 * @param htim: 定时器句柄
 * @note 由 HAL 库自动调用，每 10ms 触发一次
 */
void HAL_TIM_PeriodElapsedCallback(TIM_HandleTypeDef *htim)
{
    if (htim->Instance == TIM6)  // 10ms 定时中断
    {
        // 1. 更新编码器数据（获取当前转速）
        Encoders_All_Update();
        
        // 2. 执行 PID 控制计算
        Motor_PID_Update_All();
        
        // 3. 执行转向环 PID
        Yaw_PID_Update();
    }
}
```

#### 2.4.2 中断优先级

| 中断源 | 优先级 | 说明 |
|--------|--------|------|
| TIM6 | 最高 | PID 控制，实时性要求最高 |
| USART DMA | 中 | 数据接收，可稍后处理 |
| 其他外设 | 低 | 耗时操作 |

### 2.5 运动控制模型

#### 2.5.1 麦克纳姆轮运动学

四轮麦克纳姆轮底盘可实现全向移动：

```c
/**
 * @brief 底盘运动控制
 * @param speed_left: 左轮速度
 * @param speed_right: 右轮速度
 * 
 * 运动学模型：
 * - 左轮 = 电机B + 电机C
 * - 右轮 = 电机A + 电机D
 */
void Car_Move(int speed_left, int speed_right)
{
    // 左侧两个轮子同速
    MotorB_SetSpeed(speed_left);   // 左前
    MotorC_SetSpeed(speed_left);   // 左后
    
    // 右侧两个轮子同速
    MotorA_SetSpeed(speed_right);  // 右前
    MotorD_SetSpeed(speed_right);   // 右后
}

// 前进：左右轮同时后退
void Car_Go(int speed)
{
    Car_Move(-speed, -speed);
}

// 后退：左右轮同时前进
void Car_Back(int speed)
{
    Car_Move(speed, speed);
}

// 左转：左轮前进，右轮后退 → 绕中心旋转
void Car_TurnLeft(int speed)
{
    Car_Move(speed, -speed);
}

// 右转：左轮后退，右轮前进
void Car_TurnRight(int speed)
{
    Car_Move(-speed, speed);
}
```

#### 2.5.2 差速转向原理

```
左转示意：
        前
   ← MotorB
   ← MotorC
   ────────
   → MotorA
   → MotorD
        后

右轮加速 + 左轮减速（或反向）= 转向
```

### 2.7 常见问题分析

| 问题 | 本质原因 | 解决方案 |
|------|----------|----------|
| 电机抖动 | Kp 过大或信号干扰 | 降低 Kp，增加软件滤波 |
| 停车过冲 | Kd 不足 | 增加 Kd 或使用斜坡减速 |
| 转向跑偏 | 左右轮机械特性差异 | 独立调试 PID 参数 |
| 编码器跳变 | EMI 干扰或接线不良 | 屏蔽线，加数字滤波 |
| 启动冲击 | 目标速度突变 | 添加斜坡启动 (Ramp) |
