## 3. 舵机

本节介绍智能物流小车中使用的舵机控制，包括：
- 串口总线舵机 (Fashion Star)
- PWM 舵机 (MG996R / MG90S)
- 多自由度机械臂控制

### 3.1 硬件资源分配

| 舵机类型 | 通信方式 | 使用的 UART | 控制引脚 |
|----------|----------|--------------|----------|
| 机械臂舵机 | UART (TTL) | UART5 | PB15 (DIR) |
| 大头 PWM 舵机 | PWM | TIM12 | CH1 |
| 其他 PWM 舵机 | PWM | TIM9/10/11 | 多个通道 |

### 3.2 串口总线舵机 (Fashion Star)

#### 3.2.1 通信协议

Fashion Star 舵机使用自定义串口协议：

```
[帧头 x2] [ID] [长度] [指令] [数据...] [校验和]
0xFF 0xFF   xx   xx     xx     xx...    xx
```

**位置控制指令**：
```c
void Servo_SendPositionCommand(uint8_t id, uint16_t position)
{
    uint8_t command[13];
    command[0] = 0xFF;  // 包头1
    command[1] = 0xFF;  // 包头2
    command[2] = id;    // 舵机ID
    command[3] = 0x09;  // 数据长度
    command[4] = 0x03;  // 指令: 写
    command[5] = 0x2A;  // 地址: 目标位置
    
    // 位置低/高字节
    command[6] = (uint8_t)(position);
    command[7] = (uint8_t)(position >> 8);
    
    // 固定速度/时间
    command[8] = 0x00;
    command[9] = 0x00;
    command[10] = 0xC4;
    command[11] = 0x09;
    
    // 校验和
    uint8_t checksum = 0;
    for (int i = 2; i < 12; i++) checksum += command[i];
    command[12] = ~checksum;
    
    HAL_UART_Transmit(&huart5, command, sizeof(command), HAL_MAX_DELAY);
}
```

**带加速度的速度控制指令**：
```c
void Servo_SendPositionAccelSpeedCommand(
    UART_HandleTypeDef *huart_handle, 
    uint8_t id, 
    uint16_t acceleration, 
    int position, 
    uint16_t speed)
{
    uint8_t command[14];
    command[0] = 0xFF; command[1] = 0xFF;
    command[2] = id;
    command[3] = 0x0A;
    command[4] = 0x03;      // WRITE
    command[5] = 0x29;      // 地址: 加速度寄存器
    
    command[6] = acceleration;
    command[7] = (uint8_t)(position & 0xFF);
    command[8] = (uint8_t)(position >> 8);
    command[9] = 0x00; command[10] = 0x00;
    command[11] = (uint8_t)(speed & 0xFF);
    command[12] = (uint8_t)(speed >> 8);
    
    // 校验和
    uint8_t checksum = 0;
    for (int i = 2; i < 13; i++) checksum += command[i];
    command[13] = ~checksum;
    
    HAL_UART_Transmit(huart_handle, command, sizeof(command), HAL_MAX_DELAY);
}
```

#### 3.2.2 精华解读

- **校验和计算**：从 ID 到最后一个数据字节求和，最后取反
- **位置范围**：0-4095，对应 0°-360°
- **速度单位**：0.732 rpm
- **加速度单位**：8.878 度/s²

#### 3.3.2 大头 PWM 舵机 (datou_pwm.c)

```c
void Datou_PWM_SetAngle(uint8_t angle)
{
    // 角度转换为 PWM 脉宽
    // 0° → 500us, 180° → 2500us
    uint16_t pulse = 500 + (angle * 2000 / 180);
    uint16_t cmp = pulse * (SystemCoreClock / 1000000) / 20; // 转换为比较值
    
    __HAL_TIM_SET_COMPARE(&htim12, TIM_CHANNEL_1, cmp);
}
```

### 3.4 环形缓冲区 (Ring Buffer)

串口通信中使用环形缓冲区避免数据丢失：

```c
typedef struct {
    uint8_t *buffer;
    uint32_t size;
    uint32_t head;  // 写入位置
    uint32_t tail;  // 读取位置
    uint32_t count; // 当前数据量
} RingBufferTypeDef;

void RingBuffer_Init(RingBufferTypeDef *rb, uint32_t size, uint8_t *buf)
{
    rb->buffer = buf;
    rb->size = size;
    rb->head = rb->tail = rb->count = 0;
}

void RingBuffer_WriteByte(RingBufferTypeDef *rb, uint8_t data)
{
    rb->buffer[rb->head++] = data;
    if (rb->head >= rb->size) rb->head = 0;
    rb->count++;
}

uint8_t RingBuffer_ReadByte(RingBufferTypeDef *rb)
{
    uint8_t data = rb->buffer[rb->tail++];
    if (rb->tail >= rb->size) rb->tail = 0;
    rb->count--;
    return data;
}
```

### 3.5 常见问题与改进建议

| 问题 | 原因 | 改进建议 |
|------|------|----------|
| 舵机抖动 | PWM 频率不对或信号干扰 | 检查 50Hz 周期，增加滤波电容 |
| 舵机不转 | 串口协议错误 | 校验和计算是否正确 |
| 角度不准 | PWM 脉宽与角度映射错误 | 校准 500-2500us 对应关系 |
| 通信丢帧 | DMA 缓冲区溢出 | 增大缓冲区，使用 IDLE 中断 |
