## 3. 舵机

> **一句话概括**：让机械臂"听指令"转动到指定角度的装置

### 3.1 想象一下

舵机就像你的**手臂**：
- 肩膀关节 → 能转很大角度
- 肘关节 → 能转一定角度
- 手指关节 → 只能转一点点

舵机就是一个**带角度反馈的电机**，你告诉它"转 90 度"，它就会转到 90 度停下来。

---

### 3.2 两种舵机：PWM vs 串口

#### PWM 舵机（MG996R 等）

**原理**：像军训喊口号
- "稍息"（PWM 高电平 1ms）→ 0°
- "立正"（PWM 高电平 1.5ms）→ 90°
- "向右看齐"（PWM 高电平 2ms）→ 180°

**代码**：
```c
void Datou_PWM_SetAngle(uint8_t angle)
{
    // 把角度转换成 PWM 脉宽
    // 0° → 500us, 180° → 2500us
    uint16_t pulse = 500 + (angle * 2000 / 180);
    
    // 转换成定时器比较值
    uint16_t cmp = pulse * (SystemCoreClock / 1000000) / 20;
    
    __HAL_TIM_SET_COMPARE(&htim12, TIM_CHANNEL_1, cmp);
}
```

#### 串口舵机（Fashion Star）

**原理**：像用对讲机喊话
- 有地址（ID），像喊"3号舵机"
- 有具体指令，像"转到 2000 位置"
- 有校验，像喊话要确认对方听懂了

**通信格式**：
```
[包头 0xFF 0xFF] [ID] [长度] [指令] [数据...] [校验和]
```

**发送位置指令**：
```c
void Servo_SendPositionCommand(uint8_t id, uint16_t position)
{
    uint8_t command[13];
    command[0] = 0xFF;  // 包头1
    command[1] = 0xFF;  // 包头2
    command[2] = id;    // 舵机 ID
    command[3] = 0x09;  // 数据长度
    command[4] = 0x03;  // 指令：写
    command[5] = 0x2A;  // 地址：目标位置
    
    // 位置转成两个字节
    command[6] = (uint8_t)(position);
    command[7] = (uint8_t)(position >> 8);
    
    // 计算校验和（就像确认对方听懂了）
    uint8_t checksum = 0;
    for (int i = 2; i < 12; i++) checksum += command[i];
    command[12] = ~checksum;  // 取反
    
    // 发送！
    HAL_UART_Transmit(&huart5, command, sizeof(command), HAL_MAX_DELAY);
}
```

### 3.4 环形缓冲区——不会丢失的数据

想象你在**录音**：
- 麦克风一直录（写入）
- 你一段一段听（读取）
- 录满了怎么办？**覆盖最早的**（环形）

```c
typedef struct {
    uint8_t *buffer;
    uint32_t size;
    uint32_t head;  // 写到哪里
    uint32_t tail;  // 读到哪里
    uint32_t count; // 当前存了多少
} RingBufferTypeDef;

// 写入一个字节
void RingBuffer_WriteByte(RingBufferTypeDef *rb, uint8_t data)
{
    rb->buffer[rb->head++] = data;
    if (rb->head >= rb->size) rb->head = 0;  // 循环！
    rb->count++;
}

// 读出一个字节
uint8_t RingBuffer_ReadByte(RingBufferTypeDef *rb)
{
    uint8_t data = rb->buffer[rb->tail++];
    if (rb->tail >= rb->size) rb->tail = 0;
    rb->count--;
    return data;
}
```

### 3.5 常见问题与应对

| 问题 | 就像... | 解决方法 |
|------|---------|----------|
| 舵机抖动 | 手臂发抖 | 检查 PWM 频率是否 50Hz，检查供电是否稳定 |
| 舵机不转 | 叫了但没反应 | 检查串口协议，校验和算对了么？ |
| 角度不准 | 让你转 90°，转了 100° | 校准：调整脉宽映射公式 |
| 通讯丢帧 | 说话太快对方没听清 | 用 DMA + 缓冲区，增大缓冲 |
