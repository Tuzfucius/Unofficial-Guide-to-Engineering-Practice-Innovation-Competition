### 3.1 舵机基本原理

#### 3.1.1 内部结构

舵机是集成式位置伺服系统，内部包含：

```
┌─────────────────────────────────────────┐
│           舵机内部结构                  │
├─────────────────────────────────────────┤
│  控制信号 → 比较器 → 驱动电路 → 电机    │
│                     ↓                   │
│                   减速齿轮               │
│                     ↓                   │
│                   输出轴                 │
│                     ↓                   │
│         ← 电位器反馈 (构成闭环) →       │
└─────────────────────────────────────────┘
```

**工作原理**：
1. 控制信号输入比较器
2. 比较目标位置与电位器反馈
3. 驱动电机正转/反转
4. 电机经减速齿轮带动输出轴
5. 电位器同步转动，达到目标位置时停止

#### 3.1.2 PWM 舵机控制原理

PWM 舵机通过 PWM 信号的高电平脉宽确定目标位置：

```
PWM 周期: 20ms (50Hz)

0°  ──╲
      ╲ 0.5ms
      ╱
1.5ms ╲─────────────────────────── 90°
      ╲
      ╲ 2.0ms
      ╲__
```

**脉宽-角度公式**：
```
脉宽(μs) = 500 + 角度 × (2000/180)
```

### 3.2 PWM 舵机驱动

#### 3.2.1 定时器配置

```c
/**
 * @brief TIM12 初始化为 50Hz PWM 输出
 * @note PSC=83, ARR=19999 → 周期 = 84×20000/84MHz = 20ms
 */
static void MX_TIM12_Init(void)
{
    htim12.Instance = TIM12;
    // 预分频：84MHz / 84 = 1MHz
    htim12.Init.Prescaler = 83;
    // 自动重装载值：1MHz / 20000 = 50Hz
    htim12.Init.Period = 19999;
    htim12.Init.CounterMode = TIM_COUNTERMODE_UP;
    
    HAL_TIM_PWM_Init(&htim12);
    
    // 配置通道参数...
}
```

#### 3.2.2 角度输出

```c
/**
 * @brief 设置 PWM 舵机角度
 * @param angle: 目标角度 (0-180°)
 * 
 * 计算步骤：
 * 1. 脉宽 = 500 + angle × (2000/180)
 * 2. 比较值 = 脉宽 × 定时器频率
 * 3. 定时器频率 = 84MHz / 84 = 1MHz → 1μs/计数
 */
void Datou_PWM_SetAngle(uint8_t angle)
{
    // 1. 角度转脉宽 (μs)
    // 0° → 500μs, 90° → 1500μs, 180° → 2500μs
    uint16_t pulse_us = 500 + (angle * 2000 / 180);

    // 2. 脉宽转比较值 (定时器 1μs 精度)
    uint16_t cmp = pulse_us;  // 1MHz → 1计数 = 1μs

    // 3. 输出 PWM
    __HAL_TIM_SET_COMPARE(&htim12, TIM_CHANNEL_1, cmp);
}
```

### 3.3 串口总线舵机

#### 3.3.1 通信协议

Fashion Star 串口舵机采用自定义二进制协议：

```
┌──────┬────┬─────┬──────┬─────────────────┬────────┐
│ 包头  │ ID │ 长度 │ 指令 │ 数据            │ 校验和 │
│0xFF  │ xx │  n   │ 0x03 │ ...             │  ~Σ   │
└──────┴────┴─────┴──────┴─────────────────┴────────┘
```

**字段说明**：

| 字段 | 字节数 | 说明 |
|------|--------|------|
| 包头 | 2 | 0xFF 0xFF |
| ID | 1 | 舵机地址 (0-253) |
| 长度 | 1 | 数据长度 |
| 指令 | 1 | 0x03=WRITE |
| 数据 | n | 寄存器地址 + 值 |
| 校验和 | 1 | ~ (ID+Length+Instruction+Data) |

**指令类型**：

| 指令 | 功能 |
|------|------|
| 0x01 | 角度查询 |
| 0x03 | 位置写入 |
| 0x04 | 速度写入 |
| 0x06 | 位置速度写入(带加速度) |

#### 3.3.2 位置控制指令

```c
/**
 * @brief 发送位置控制指令
 * @param id: 舵机 ID (0-253)
 * @param position: 目标位置 (0-4095，对应 0°-360°)
 * 
 * 指令格式：
 * [FF FF] [ID] [09] [03] [2A] [PosL PosL] [PosH] [00 00] [SpdL SpdH] [CS]
 *           │    │    │    │    │        │        │
 *           │    │    │    │    │        └────────┴────── 速度
 *           │    │    │    │    └─────────────────────── 时间(未用)
 *           │    │    │    └─────────────────────────── 位置(2字节)
 *           │    │    └──────────────────────────────── 地址(0x2A=目标位置)
 *           │    └───────────────────────────────────── 指令(WRITE)
 *           └─────────────────────────────────────────── 数据长度(9)
 */
void Servo_SendPositionCommand(uint8_t id, uint16_t position)
{
    uint8_t command[13];

    // 帧头
    command[0] = 0xFF;
    command[1] = 0xFF;

    // 地址与控制
    command[2] = id;        // 舵机 ID
    command[3] = 0x09;     // 数据长度 9
    command[4] = 0x03;     // 指令：WRITE
    command[5] = 0x2A;     // 地址：目标位置寄存器 (42)

    // 数据：位置 (低字节在前)
    command[6] = (uint8_t)(position & 0xFF);
    command[7] = (uint8_t)((position >> 8) & 0xFF);
    command[8] = 0x00;     // 时间低字节
    command[9] = 0x00;     // 时间高字节
    command[10] = 0xC4;    // 速度低字节
    command[11] = 0x09;    // 速度高字节

    // 校验和计算：~ (ID + Length + Instruction + Address + Data)
    uint8_t checksum = 0;
    for (int i = 2; i < 12; i++) checksum += command[i];
    command[12] = ~checksum;  // 取反

    // 发送
    HAL_UART_Transmit(&huart5, command, 13, HAL_MAX_DELAY);
}
```

#### 3.3.3 带加速度的速度控制

```c
/**
 * @brief 带加速度的位置控制
 * @param acceleration: 加速度 (0-255, 8.878°/s²)
 * @param position: 目标位置 (0-4095)
 * @param speed: 最大速度 (0-3400, 0.732rpm)
 */
void Servo_SendPositionAccelSpeedCommand(
    UART_HandleTypeDef *huart,
    uint8_t id,
    uint16_t acceleration,
    int position,
    uint16_t speed)
{
    uint8_t command[14];

    command[0] = 0xFF; command[1] = 0xFF;
    command[2] = id;
    command[3] = 0x0A;     // 数据长度 10 (新增加速度)
    command[4] = 0x03;     // WRITE
    command[5] = 0x29;     // 地址：加速度寄存器 (41)

    command[6] = acceleration;
    command[7] = (uint8_t)(position & 0xFF);
    command[8] = (uint8_t)((position >> 8) & 0xFF);
    command[9] = 0x00; command[10] = 0x00;  // 时间
    command[11] = (uint8_t)(speed & 0xFF);
    command[12] = (uint8_t)((speed >> 8) & 0xFF);

    uint8_t checksum = 0;
    for (int i = 2; i < 13; i++) checksum += command[i];
    command[13] = ~checksum;

    HAL_UART_Transmit(huart, command, 14, HAL_MAX_DELAY);
}
```

### 3.4 环形缓冲区

#### 3.4.1 原理

串口数据逐字节到达，使用环形缓冲区实现异步数据缓存：

```
┌─────────────────────────────────────┐
│  buffer[0] ... buffer[head] ...   │
│                  ↑                   │
│                  tail               │
└─────────────────────────────────────┘

写入：head++ (到末尾则回绕到 0)
读取：tail++ (到末尾则回绕到 0)
```

#### 3.4.2 实现

```c
/**
 * @brief 环形缓冲区结构体
 */
typedef struct {
    uint8_t *buffer;    // 数据存储区
    uint32_t size;      // 缓冲区总大小
    uint32_t head;      // 写入位置
    uint32_t tail;      // 读取位置
    uint32_t count;     // 当前数据个数
} RingBufferTypeDef;

/**
 * @brief 写入一个字节
 */
void RingBuffer_WriteByte(RingBufferTypeDef *rb, uint8_t data)
{
    rb->buffer[rb->head++] = data;
    if (rb->head >= rb->size) rb->head = 0;  // 回绕
    rb->count++;
}

/**
 * @brief 读取一个字节
 */
uint8_t RingBuffer_ReadByte(RingBufferTypeDef *rb)
{
    uint8_t data = rb->buffer[rb->tail++];
    if (rb->tail >= rb->size) rb->tail = 0;
    rb->count--;
    return data;
}
```

### 3.6 常见问题分析

| 问题 | 本质原因 | 解决方案 |
|------|----------|----------|
| 舵机抖动 | PWM 频率偏差或信号干扰 | 校准 50Hz，增加滤波电容 |
| 不转动 | 协议错误或供电不足 | 校验和重算，检查电源 |
| 角度误差 | 脉宽映射偏差 | 实际测量校准公式 |
| 通信丢帧 | DMA 缓冲区不足 | 增大缓冲区，使用 IDLE 中断 |
