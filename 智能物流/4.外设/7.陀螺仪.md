## 7. 陀螺仪 (HWT101)

> **一句话概括**：让小车知道"朝向哪"——像指南针 + 计步器的组合

### 7.1 想象一下

你闭着眼站在房间里：
- **陀螺仪**告诉你：头转了多少度（偏航角）
- **加速度计**告诉你：往哪边倾斜了

HWT101 就是一个**测量角度变化**的传感器，帮助小车知道"现在朝向哪里"。

---

### 7.2 数据怎么传？

HWT101 用**二进制格式**发数据，像发电报：

```
[包头 0x55] [类型] [数据 10 字节] [校验和]
0x55       0x53   XX...          0xXX
```

常见类型：
- `0x52`：角速度（转得快不快）
- `0x53`：角度（转了多少度）

---

### 7.3 STM32 怎么接收？

和 OpenMV 类似，用 **DMA + 空闲中断**：

```c
void HWT101_Init(UART_HandleTypeDef *huart)
{
    // 1. 发送校准指令
    uint8_t cmd[] = {0xFF, 0xAA, 0x76, 0x00, 0x00};
    HAL_UART_Transmit(huart, cmd, sizeof(cmd), 100);
    
    HAL_Delay(20);
    
    // 2. 启动 DMA 接收
    HAL_UARTEx_ReceiveToIdle_DMA(huart, rx_buffer, RX_BUFFER_SIZE);
}
```

### 7.4 数据解析——破译电报

```c
void HWT101_ParseData(const uint8_t* data, uint16_t size)
{
    for (uint16_t i = 0; i < size; i++)
    {
        // 1. 找包头
        if (data[i] == 0x55)
        {
            if (i + 11 > size) continue;  // 数据不完整
            
            // 2. 校验和
            uint8_t sum = 0;
            for (int j = 0; j < 10; j++) sum += data[i+j];
            if (sum != data[i+10]) continue;  // 校验失败
            
            // 3. 解析
            if (data[i+1] == 0x53)  // 角度包
            {
                int16_t raw = (data[i+7] << 8) | data[i+6];
                // 原始值 → 实际角度
                float angle = (float)raw / 32768.0f * 180.0f;
                
                // 减去零偏（校准）
                g_hwt101_data.Angle_Z = angle - g_angle_offset;
            }
        }
    }
}
```

**公式解释**：
- `32768` = 2^15，因为用 15 位表示有符号数
- `/ 32768 * 180`：把范围 [-32768, 32767] 映射到 [-180°, 180°]

### 7.5 零偏校准——每次开机归零

就像每次用指南针前要**校准**，陀螺仪也需要：

```c
void HWT101_ZeroYaw(void)
{
    // 记录当前角度作为"零点"
    g_angle_offset = g_latest_raw_angle_deg;
    g_hwt101_data.Angle_Z = 0.0f;
}
```

**这就好像**：你站在门口，面向大门，按一下"归零"，以后所有角度都相对于这个朝向。

---

### 7.6 角度归一化——处理"过零点"

想象转圈：
- 从 179° → 180° → -179° → -178° ...
- **不能直接相减**！179 - (-179) = 358（错了！）

**正确做法**：
```c
float normalize_angle(float angle)
{
    while (angle > 180)  angle -= 360;
    while (angle < -180) angle += 360;
    return angle;
}
```

### 7.7 在转向控制中的应用

```c
void Yaw_PID_Update(void)
{
    float current = g_hwt101_data.Angle_Z;  // 当前朝向
    float target = Yaw_PID.Target;          // 目标朝向
    
    // 计算误差（考虑 180° 过零点）
    float error = target - current;
    if (error > 180)  error -= 360;
    if (error < -180) error += 360;
    
    // PID 计算...
    float output = PID_Calculate(&Yaw_PID, 0, error);
    
    // 输出：左轮快右轮慢 = 转向
    Car_Move(MotorBaseSpeed - output, MotorBaseSpeed + output);
}
```

### 7.8 常见问题与应对

| 问题 | 就像... | 解决方法 |
|------|---------|----------|
| 角度漂移 | 计步器自己会跳 | 启动时校准，减少长时间积分 |
| 数据跳动 | 指南针乱晃 | 增加滤波（卡尔曼滤波） |
| 初始化失败 | 叫了但没人理 | 检查串口波特率 115200 |
| 角度不准 | 感觉方向偏了 | 启动时调用 ZeroYaw() |

### 7.10 通信要点

- **波特率**：115200
- **数据位**：8
- **停止位**：1
- **校验位**：无
- **协议**：二进制自定义格式
- **量程**：±2000°/s（角速度），±180°（角度）
