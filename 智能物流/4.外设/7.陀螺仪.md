## 7. 陀螺仪 (HWT101)

本节介绍智能物流小车中使用的 HWT101 陀螺仪模块，用于获取小车偏航角（航向角）。

### 7.1 硬件资源分配

| 设备 | 通信方式 | 使用的 UART | 备注 |
|------|----------|--------------|------|
| HWT101 | UART (DMA + IDLE) | UART4 | 115200bps, 8N1 |

### 7.2 通信协议

HWT101 使用二进制协议，数据包格式：

```
[包头 0x55] [数据类型] [数据...] [校验和]
0x55       0xXX        XX...      0xXX
```

**常用数据包类型**：
- `0x52`: 角速度包 (Angular Velocity)
- `0x53`: 角度包 (Angle)

### 7.3 数据结构

```c
typedef struct {
    float Angle_X;      // X 轴角度 (俯仰)
    float Angle_Y;      // Y 轴角度 (横滚)
    float Angle_Z;      // Z 轴角度 (偏航/航向)
    float AngVel_X;     // X 轴角速度
    float AngVel_Y;     // Y 轴角速度
    float AngVel_Z;     // Z 轴角速度
    int16_t Temp;       // 温度
} HWT101_Data_t;

HWT101_Data_t g_hwt101_data;
```

### 7.4 初始化与校准

```c
void HWT101_Init(UART_HandleTypeDef *huart)
{
    hwt101_huart = huart;
    
    // 清零数据结构
    memset(&g_hwt101_data, 0, sizeof(HWT101_Data_t));
    
    // 发送 Z 轴角度清零指令
    uint8_t reset_z_axis_cmd[] = {0xFF, 0xAA, 0x76, 0x00, 0x00};
    HAL_UART_Transmit(hwt101_huart, reset_z_axis_cmd, 
                      sizeof(reset_z_axis_cmd), 100);
    
    HAL_Delay(20);
    
    // 启动 DMA + IDLE 接收
    HAL_UARTEx_ReceiveToIdle_DMA(hwt101_huart, rx_buffer, RX_BUFFER_SIZE);
}
```

**精华解读**：
- 发送 `0xFF 0xAA 0x76 0x00 0x00` 校准 Z 轴
- 使用 DMA + IDLE 中断实现不定长数据接收

### 7.5 数据解析

```c
void HWT101_ParseData(const uint8_t* pData, uint16_t Size)
{
    for (uint16_t i = 0; i < Size; ++i)
    {
        // 1. 寻找包头 0x55
        if (pData[i] == 0x55)
        {
            if (i + 11 > Size) continue; // 数据不完整
            
            // 2. 校验和验证
            uint8_t sum = 0;
            for (int j = 0; j < 10; ++j)
            {
                sum += pData[i + j];
            }
            
            if (sum != pData[i + 10]) continue; // 校验失败
            
            // 3. 解析数据
            switch (pData[i + 1])
            {
                case 0x52: // 角速度包
                {
                    int16_t data_raw;
                    data_raw = (int16_t)((pData[i + 5] << 8) | pData[i + 4]);
                    g_hwt101_data.AngVel_Y = (float)data_raw / 32768.0f * 2000.0f;
                    
                    data_raw = (int16_t)((pData[i + 7] << 8) | pData[i + 6]);
                    g_hwt101_data.AngVel_Z = (float)data_raw / 32768.0f * 2000.0f;
                    break;
                }
                case 0x53: // 角度包
                {
                    int16_t data_raw;
                    data_raw = (int16_t)((pData[i + 7] << 8) | pData[i + 6]);
                    g_latest_raw_angle_deg = (float)data_raw / 32768.0f * 180.0f;
                    g_hwt101_data.Angle_Z = normalize_angle_deg(
                        g_latest_raw_angle_deg - g_angle_z_offset_deg);
                    break;
                }
            }
        }
    }
}
```

**精华解读**：
- **数据转换公式**：`实际值 = 原始值 / 32768.0 * 量程`
- **角速度量程**：默认 ±2000°/s
- **角度量程**：默认 ±180°
- **校验和**：前 10 字节之和应等于第 11 字节

### 7.6 角度归一化

```c
static float normalize_angle_deg(float angle)
{
    while (angle > 180.0f)  angle -= 360.0f;
    while (angle < -180.0f) angle += 360.0f;
    return angle;
}
```

**精华解读**：
- 将角度归一化到 [-180°, +180°] 范围
- 便于转向控制和角度比较

### 7.7 零偏校准

```c
void HWT101_ZeroYaw(void)
{
    g_angle_z_offset_deg = g_latest_raw_angle_deg;
    g_hwt101_data.Angle_Z = 0.0f;
}
```

### 7.8 串口空闲中断回调

```c
void HAL_UARTEx_RxEventCallback(UART_HandleTypeDef *huart, uint16_t Size)
{
    if (huart->Instance == UART4)
    {
        // 解析数据
        HWT101_ParseData(rx_buffer, Size);
        
        // 清空缓冲区
        memset(rx_buffer, 0, Size);
        
        // 重新启动 DMA 接收
        HAL_UARTEx_ReceiveToIdle_DMA(&huart4, rx_buffer, RX_BUFFER_SIZE);
    }
}
```

### 7.10 常见问题与改进建议

| 问题 | 原因 | 改进建议 |
|------|------|----------|
| 角度漂移 | 积分误差累积 | 使用硬件校准，减少积分时间 |
| 数据跳动 | 校验失败或干扰 | 增加校验，增加滤波（如卡尔曼滤波） |
| 初始化失败 | 串口配置错误 | 检查波特率 115200，数据位 8，校验 N |
| 角度不准确 | 未零偏校准 | 启动时调用 `HWT101_ZeroYaw()` |
